# Getting Started

## 1. Download and Install

Go to the [Download page](/en/download) to get the installer for your operating system.

## 2. Import Server

After launching Sea Lantern, import a Minecraft server JAR file.

## 3. Select Java

Choose the Java runtime installed on your system. If you don't have a suitable version, use the built-in Java downloader to install one with a single click.

## 4. Start Server

Click the start button. It's that simple.

## 5. Personalize

In the Settings page, you can:

- Switch the interface language (12 languages supported)
- Choose a theme and dark/light mode
- Enable acrylic effect
