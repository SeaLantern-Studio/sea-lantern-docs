# Features

## Console

View server log output in real-time and enter commands directly in the console. Supports command history, quick command panel, and toolbar actions.

## server.properties Editor

Edit server configuration with a graphical interface. Supports categorized browsing and search filtering — no need to manually modify files.

## Player Management

- Whitelist management
- Ban management
- OP permission management

## Plugin System

Lua script-based plugin extension mechanism:

- Plugins can register custom UI components and pages
- Context menu extensions
- Plugin permission management panel
- Sandboxed plugin runtime

## Theme System

- 5 built-in themes: Default, Midnight, Ocean, Rose, Sunset
- Dark/light mode toggle
- Acrylic/frosted glass effect
- Complete CSS Variables design token system with runtime switching

## Multi-language Support

12 built-in languages:

- 简体中文, 繁體中文
- English, 日本語, 한국어
- Français, Deutsch, Español
- Русский, Tiếng Việt

## Java Management

- Auto-detect Java runtimes installed on the system
- Built-in Java downloader for one-click installation

## Mod Management

View and manage server-side mods/plugins.

## Server Creation

Guided server creation workflow for quickly configuring and launching a new Minecraft server.

## Safe Shutdown

Sea Lantern automatically stops the Minecraft server safely when closing, preventing save corruption.

## Auto Update

Check for new versions with incremental download and checksum verification. One-click install.

## System Tray

Minimize to system tray and keep the server running in the background.
