# 快速開始

## 1. 下載安裝

前往 [下載頁面](/zh-tw/download) 取得適合您作業系統的安裝包。

## 2. 匯入伺服端

啟動 Sea Lantern 後，匯入一個 Minecraft 伺服端 JAR 檔案。

## 3. 選擇 Java

選擇您系統中已安裝的 Java 執行環境。如果沒有合適的版本，可以使用內建的 Java 下載器一鍵安裝。

## 4. 啟動伺服器

點擊啟動按鈕，就這麼簡單。

## 5. 個人化設定

在設定頁面中，您可以：

- 切換介面語言（支援 12 種語言）
- 選擇主題和明暗模式
- 開啟亞克力效果
