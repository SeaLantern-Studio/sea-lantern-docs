# 功能總覽

本頁用於快速了解 Sea Lantern 目前可用能力與適用場景。  
如果你是首次使用，建議先看「伺服器建立」「Java 管理」「控制台」三個區塊。

## 控制台

- 即時查看伺服器日誌輸出，涵蓋啟動與運行階段
- 直接輸入命令，支援命令歷史與快捷命令面板
- 工具列提供清空顯示、匯出日誌等常用操作

## server.properties 編輯器

- 圖形化介面編輯伺服器配置，不需要手動改檔
- 支援分類瀏覽與搜尋篩選，定位配置更快
- 適合日常參數調整；部分項目仍需重啟伺服器生效

## 玩家管理

- 線上玩家狀態檢視
- 白名單管理
- 封禁管理
- OP 權限管理
- 支援踢出玩家等常見管理操作

## 插件系統

基於 Lua 腳本的插件擴展機制：

- 插件可註冊自訂 UI 元件和頁面
- 支援右鍵選單擴展
- 插件權限管理面板
- 插件執行時沙箱隔離
- 支援插件安裝、啟用/停用與刪除

## 主題系統

- 5 套內建主題：預設、午夜、海洋、玫瑰、日落
- 支援明暗模式切換
- 亞克力/毛玻璃效果（Windows）
- 基於 CSS Variables 的完整設計令牌體系，支援執行時切換

## 多語言支援

內建 10 種語言：

- 簡體中文、繁體中文
- English、日本語、한국어
- Français、Deutsch、Español
- Русский、Tiếng Việt
- 支援執行時切換語言，無需重啟應用

## Java 管理

- 自動偵測系統中已安裝的 Java 執行環境
- 內建 Java 下載器，支援一鍵安裝所需版本
- 支援手動指定 Java 路徑並進行可用性校驗

## Mod 管理

- 檢視伺服端目錄中已安裝的 Mod/插件檔案
- 便於排查版本衝突、重複安裝與目錄問題
- 目前以本地管理為主，線上檢索與一鍵安裝能力仍在持續完善

## 伺服器建立

- 引導式流程建立新伺服器，降低新手配置門檻
- 支援 `jar` / `bat` / `sh` 啟動模式
- 首次啟動可引導完成 EULA 相關步驟

## 安全退出

- 關閉 Sea Lantern 時自動觸發安全停止流程
- 盡量避免強制退出造成的存檔損壞風險

## 自動更新

- 檢查新版本並顯示目前版本與最新版本資訊
- 提供版本說明查看入口與下載跳轉
- Arch Linux 使用 AUR 方式更新，其它平台走各自安裝流程

## 系統匣

- 最小化到系統匣後可保持背景運行
- 支援從系統匣快速還原視窗或執行退出

## 使用建議

- 新服首次上線：先用「伺服器建立」完成初始化，再用「圖形化配置」調整關鍵參數
- 日常維運：優先透過「控制台」與「玩家管理」完成高頻操作，減少手寫命令
- 長時間掛服：搭配「系統匣」與「安全退出」降低誤操作與強退風險

## 功能邊界說明

- 不同伺服端核心、不同作業系統的行為可能有差異，建議依各頁說明配置
- 自動更新以「檢查版本 + 引導下載」為主，是否覆蓋安裝由你自行確認
