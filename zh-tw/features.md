# 功能總覽

## 控制台

即時查看伺服器日誌輸出，直接在控制台輸入命令。支援命令歷史記錄、快捷命令面板和工具列操作。

## server.properties 編輯器

圖形化介面編輯伺服器配置，支援分類瀏覽、搜尋篩選，不需要手動修改檔案。

## 玩家管理

- 白名單管理
- 封禁管理
- OP 權限管理

## 插件系統

基於 Lua 腳本的插件擴展機制：

- 插件可註冊自訂 UI 元件和頁面
- 支援右鍵選單擴展
- 插件權限管理面板
- 插件執行時沙箱隔離

## 主題系統

- 5 套內建主題：預設、午夜、海洋、玫瑰、日落
- 支援明暗模式切換
- 亞克力/毛玻璃效果
- 基於 CSS Variables 的完整設計令牌體系，支援執行時切換

## 多語言支援

內建 12 種語言：

- 簡體中文、繁體中文
- English、日本語、한국어
- Français、Deutsch、Español
- Русский、Tiếng Việt

## Java 管理

- 自動偵測系統中已安裝的 Java 執行環境
- 內建 Java 下載器，支援一鍵安裝所需版本

## Mod 管理

檢視和管理伺服端已安裝的 Mod/插件。

## 伺服器建立

引導式伺服器建立流程，快速配置並啟動新的 Minecraft 伺服器。

## 安全退出

關閉 Sea Lantern 時會自動安全停止 Minecraft 伺服器，防止存檔損壞。

## 自動更新

檢查新版本，支援增量下載和校驗，一鍵安裝最新版。

## 系統匣

最小化到系統匣，背景保持伺服器執行。
