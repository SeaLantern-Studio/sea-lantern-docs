# 快速开始

## 1. 下载安装

前往 [下载页面](/zh/download) 获取适合你操作系统的安装包。

## 2. 导入服务端

启动 Sea Lantern 后，导入一个 Minecraft 服务端 JAR 文件。

## 3. 选择 Java

选择你系统中已安装的 Java 运行时。如果没有合适的版本，可以使用内置的 Java 下载器一键安装。

## 4. 启动服务器

点击启动按钮，就这么简单。

## 5. 个性化设置

在设置页面中，你可以：

- 切换界面语言（支持 12 种语言）
- 选择主题和明暗模式
- 开启亚克力效果
