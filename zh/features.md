# 功能总览

## 控制台

实时查看服务器日志输出，直接在控制台输入命令。支持命令历史记录、快捷命令面板和工具栏操作。

## server.properties 编辑器

图形化界面编辑服务器配置，支持分类浏览、搜索筛选，不需要手动修改文件。

## 玩家管理

- 白名单管理
- 封禁管理
- OP 权限管理

## 插件系统

基于 Lua 脚本的插件扩展机制：

- 插件可注册自定义 UI 组件和页面
- 支持右键菜单扩展
- 插件权限管理面板
- 插件运行时沙箱隔离

## 主题系统

- 5 套内置主题：默认、午夜、海洋、玫瑰、日落
- 支持明暗模式切换
- 亚克力/毛玻璃效果
- 基于 CSS Variables 的完整设计令牌体系，支持运行时切换

## 多语言支持

内置 12 种语言：

- 简体中文、繁体中文
- English、日本語、한국어
- Français、Deutsch、Español
- Русский、Tiếng Việt

## Java 管理

- 自动检测系统中已安装的 Java 运行时
- 内置 Java 下载器，支持一键安装所需版本

## Mod 管理

查看和管理服务端已安装的 Mod/插件。

## 服务器创建

引导式服务器创建流程，快速配置并启动新的 Minecraft 服务器。

## 安全退出

关闭 Sea Lantern 时会自动安全停止 Minecraft 服务器，防止存档损坏。

## 自动更新

检查新版本，支持增量下载和校验，一键安装最新版。

## 系统托盘

最小化到系统托盘，后台保持服务器运行。
